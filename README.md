# Installation


// Download the script
```
wget https://raw.githubusercontent.com/sangrepollo/ramon/main/install_namydev.sh
```
// Edit script configuration 
```
nano install_namydev.sh
```
// Install the script
```
chmod +x install_namydev.sh; ./install_namydev.sh
```
